# Security Update — Anti-ban

## Изменения

### 🛡 Рассылка (главная причина банов)
- Задержка между сообщениями: **5–15с → 35–90с**
- Длинная пауза каждые 5 сообщений: **+90–180с** (батч-пауза)
- Дневной лимит: **40 сообщений** с одного аккаунта
- При `PeerFloodError`: рассылка немедленно останавливается + уведомление в бот
- Имитация онлайна (`UpdateStatusRequest`) перед каждой отправкой
- Более реалистичный `typing` action: 2–4.5с вместо 1–3с

### 🔒 Fingerprint устройств
- Раньше все аккаунты представлялись как "Galaxy S25 Ultra / Android 15"
- Теперь: пул из 7 реальных устройств, каждый аккаунт получает уникальный fingerprint

### 🤖 Автоответчик
- Cooldown 120с между авто-ответами одному чату
- Предотвращает спам-петли

### 🔔 Уведомления
- При деавторизации аккаунта (бан сессии) — уведомление владельцу в бот
- При `PeerFloodError` — уведомление с рекомендацией подождать 24ч

### 📊 UI рассылки
- Показывает дневной лимит и остаток в меню
- Предупреждение при > 30 получателей
- Кнопка "Начать" заблокирована если лимит исчерпан
